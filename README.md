# Website Feedback AI

This project analyzes website user comments and generates insights using SQL and AI.

## Workflow
1. Store raw comments
2. Clean and deduplicate data
3. Classify sentiment and topics
4. Generate reports
5. Summarize using AI

## Features
- Sentiment analysis
- Topic classification
- User segmentation (gender, city type)
- Page-level issue detection
- Automated summarization

## Tech Stack
- MySQL
- Python
- NLP (Transformers)

## Output
- Daily summary report
- Top issues
- User insights
